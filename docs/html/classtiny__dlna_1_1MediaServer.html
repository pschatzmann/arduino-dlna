Detailed Description

Minimal Digital Media Server implementation.

This class implements a lightweight DLNA MediaServer device with a ContentDirectory service (Browse) and a ConnectionManager service. The implementation is intentionally compact and suitable as a starting point for embedding content lists. It returns a single test item for Browse requests.
